Watchtower — quick help

  • Are.na channel — Pick where new files go. Use Refresh after arena login. You can Create channel here; when it succeeds, that channel becomes the upload target.
  • CLI — Run arena commands (read-only flow, same idea as Terminal). Output appears in the box below; Use clear next to the CLI heading.
  • Watcher — In Preferences (⌘, on Mac or File → Preferences), choose a watch folder (and options). Then use Start watcher / Stop watcher here. New files in that folder upload to the selected channel via arena.
  • Activity — Log of uploads, paths, and errors. Use clear next to the Activity heading.
  • Check login (whoami) — Confirms the CLI sees your Are.na account.

Preferences also set theme, open at login, Trash-after-upload prompts, and language.
